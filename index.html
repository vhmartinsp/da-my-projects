Projeto de Web Crawling e Scraping de Criptomoedas

Uma análise de criptomoedas no Yahoo Finance usando técnicas de Web Crawling e Scraping.

Descrição do Projeto

Este projeto é uma investigação abrangente de todas as criptomoedas listadas no Yahoo! Por meio de web crawling e scraping, colhemos informações relevantes e as apresentamos de forma organizada que permite a análise e a tomada de decisão de investimentos.

Resultados

Toda a extração foi feita diretamente do site do Yahoo Finance e vocês podem conferir as informações mais valiosas que tirei aqui de todo o processo.

Usando uma seleção de bibliotecas cuidadosamente escolhidas como re, BeautifulSoup, Path, matplotlib, pandas, explorei a coleta e análise de informações sobre criptomoedas, além de fazer uma solicitação HTTP para o URL do Yahoo Finance para obter dados atualizados sobre criptomoedas.

Tratamento e Análise de Dados: Usando a biblioteca BeautifulSoup, analisei o conteúdo da página da web e extraí as informações necessárias. Depois disso, formatei os valores e criei um DataFrame usando a biblioteca pandas. Durante minha análise, me aprofundei nas mudanças percentuais positivas do dia de criptomoedas, realizando o filtro para pegar justamente as valorizadas daquele momento.

Conclusões da Análise: Durante minha investigação, me deparei com informações bem interessantes:

  • Diversidade de Desempenho: Criptomoedas com diferentes valores de mercado apresentaram mudanças percentuais positivas. Isso sugere que o desempenho não é estritamente dependente do valor de mercado, proporcionando oportunidades para diversos investidores.
  • Oportunidades de Investimento: Muitas criptomoedas, independentemente do valor de mercado, demonstraram potencial para mudanças percentuais positivas. Isso pode ser uma oportunidade intrigante para investidores em busca de ativos promissores.
  • Abordagem Estratégica: A análise ressalta a importância da diversificação em um mercado volátil. Investidores podem considerar criptomoedas com diferentes valores de mercado para mitigar riscos.
