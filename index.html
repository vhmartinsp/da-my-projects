Projeto de Web Crawling e Scraping para Análise de Criptomoedas no Yahoo Finance

Projeto de Web Crawling e Scraping para puxar informações de uma tabela presente em HTML assim como retornar ela em um DataFrame com informações úteis, além de um gráfico de dispersão que exibe a correlação entre a variação percentual e o valor de mercado de cada criptomoeda.
